# 도감 꾸미기 아이템 단계 문서

## 목표

사용자가 `고양이 서랍`에서 꾸미기 아이템을 선택하면 도감 표지와 미리보기가 즉시 바뀌어야 한다. 무료 사용자의 핵심 수집 흐름은 유지하고, `냥꾸러미`는 더 많은 표현 선택지를 제공한다.

## 아이템 우선순위

1. 표지 테마
   - 도감 전체 분위기를 바꾸는 가장 큰 시각 요소다.
   - 선택 즉시 `고양이 서랍` 미리보기와 도감 화면 상단 표지에 반영한다.
   - 무료 기본 테마와 `냥꾸러미` 프리미엄 테마를 함께 노출한다.

2. 골목 배지 진열
   - 사용자가 획득한 배지를 도감 표지에 직접 붙이는 단계다.
   - 무료 사용자는 최대 2개, `냥꾸러미` 사용자는 최대 4개까지 진열한다.
   - 획득하지 않은 배지는 설명만 보여주고 진열할 수 없게 한다.

3. 냥발 도장
   - 시즌 참여감과 기간 한정 수집감을 만든다.
   - MVP에서는 획득/잠금 상태를 보여주고, 다음 단계에서 표지 진열 슬롯을 추가한다.
   - 시즌 기간이 지난 도장은 클라이언트 직접 insert가 아니라 RPC로만 획득하게 한다.

4. 우리 도감 주인공
   - 무료 사용자는 1마리, `냥꾸러미` 사용자는 3마리까지 표지에 노출한다.
   - 이미 수집한 고양이만 선택할 수 있어야 한다.

5. 표지 스티커와 프레임
   - 표지 테마와 배지 진열이 안정화된 뒤 추가한다.
   - 예: 발자국 스티커, 이름표 프레임, 공유 카드 테두리.

## 감성 세트

직접적인 특정 스튜디오/작가 스타일명은 쓰지 않고, 앱 안에서는 아래처럼 독자적인 감성 이름으로 제공한다.

- 계절별 감성: `봄꽃 골목`, `여름 처마 그늘`, `가을 벽돌길`, `겨울 담요`
- 따뜻한 손그림 감성: `손그림 숲길`, `구름빵 골목`, `나무집 창가`
- 아기자기한 감성: `스티커 서랍`, `리본 생선`, `파스텔 발자국`
- 중유럽 감성: `프라하 지붕`, `작은 광장 카페`, `오래된 돌길`
- 밤 산책 감성: `달빛 골목`, `별빛 지붕`, `가로등 아래`

## 단계별 구현

### 1단계: 표지 테마와 골목 배지 진열

- 감성별 표지 테마 seed를 늘린다.
- 배지 seed를 늘린다.
- `collection_profiles.selected_badge_ids`를 실제 도감 표지에 반영한다.
- 사용자가 배지를 누르면 진열/해제가 저장되고, 화면을 다시 불러온 뒤 도감 표지가 바뀐다.

검증:

- 상태: 구현 완료
- `npm run typecheck` 통과 필요
- 무료 사용자는 획득 배지 2개까지만 진열 가능
- `냥꾸러미` 사용자는 획득 배지 4개까지 진열 가능
- 획득하지 않은 배지는 진열되지 않음
- 도감 화면의 표지에 선택한 배지명이 보임

### 2단계: 냥발 도장 진열 슬롯

- 상태: 구현 중
- `collection_profiles.selected_stamp_ids`에 선택한 도장 ID 목록을 둔다.
- 획득 여부와 구독 상태를 DB 함수에서 검증한다.
- 무료 사용자는 1개, `냥꾸러미` 사용자는 3개까지 표지에 표시한다.
- 시즌 도장 임의 insert 권한은 제거하고, 실제 획득 판정은 후속 RPC/Edge Function에서 처리한다.

검증:

- 현재 시즌 도장만 획득 가능
- 프리미엄 도장은 `냥꾸러미` 사용자만 획득/진열 가능
- 기간이 지난 도장은 임의 획득되지 않음

### 3단계: 표지 스티커와 프레임

- 범용 `collection_items` 테이블을 추가한다.
- item type은 `sticker`, `frame`, `nameplate`로 시작한다.
- 표지 위에 과하게 겹치지 않는 고정 슬롯 방식으로 배치한다.

검증:

- 작은 화면에서 스티커와 텍스트가 겹치지 않음
- 무료/구독 제한이 저장 단계에서 같이 검증됨

## 현재 단계

현재 작업은 2단계다. 표지 테마와 골목 배지 진열은 화면 반영까지 연결했고, 냥발 도장 진열 슬롯을 도감 표지와 고양이 서랍 미리보기에 연결한다.
