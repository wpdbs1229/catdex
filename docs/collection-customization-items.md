# 도감 표지 기능 제거 기록

도감 화면에는 더 이상 별도의 표지 영역을 두지 않는다. 이전에 표지 테마, 표지 배지 진열, 냥발 도장 진열, 표지 스티커/프레임으로 분리했던 꾸미기 기능은 제품 범위에서 제외한다.

현재 유지하는 항목은 대표 고양이뿐이다. 대표 고양이는 표지가 아니라 홈과 사원증/마이페이지에서 사용자의 최근 관심 고양이를 보여주는 보조 정보로만 사용한다.

구현 기준:

- 도감 화면 첫 영역은 바로 고양이 목록과 검색/필터다.
- `collection_themes`, `collection_profiles`, `season_stamps`, `user_season_stamps`는 제거 마이그레이션에서 정리한다.
- 획득 배지는 배지북에서만 확인한다.
- 대표 고양이 설정 배지는 `featured_cats` 기준으로만 판단한다.
