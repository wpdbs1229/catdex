# 냥도감 전체 앱 QA 기준서

## 목적

- 현재 `main` 기준으로 사용자가 접근할 수 있는 모든 화면과 세부 기능을 나열한다.
- UI 표시, 화면 전환, Supabase 데이터 계약, 권한, 로딩, 빈 상태, 실패 복구를 함께 검증한다.
- 오류를 발견하면 같은 항목에 재현 조건, 수정 내용, 재검증 결과를 기록한다.
- 스크린샷만으로 확인할 수 없는 RLS, Storage 정리, RPC 원자성은 별도 DB 검증으로 확인한다.

## 상태 표기

- `대기`: 아직 실행하지 않음
- `통과`: 정상·예외 흐름을 실행해 기대 결과와 일치함
- `수정`: 결함을 확인해 코드 또는 마이그레이션 수정 중
- `재검증`: 수정 후 같은 흐름을 다시 확인해야 함
- `제한`: 시뮬레이터 또는 외부 서비스 제약으로 일부만 확인함

## 검증 환경

- 클라이언트: Expo React Native + TypeScript
- 플랫폼: iOS Simulator 개발 빌드, 작은 화면 우선
- 데이터: 연결된 원격 Supabase Auth, Postgres, Storage, RPC
- 정적 검사: `npm run typecheck`, `npx expo-doctor`, `git diff --check`
- 시각 검사: 화면별 현재 실행 스크린샷, 접근성 요소 스냅샷
- DB 검사: 인증 사용자/다른 사용자/익명 역할별 SQL 및 Data API 계약

## 기능 인벤토리

### 1. 앱 시작과 인증

- 스플래시: 앱 시작 표시, 저장 세션 복원 대기, 1.4초 최소 노출.
- 세션 복원: SecureStore 세션 파싱, Supabase 토큰 갱신, 만료 세션 삭제, 로그인 화면 복귀.
- 로그인: 카카오 OAuth, Google OAuth, 중복 탭 방지, 진행 상태, 취소·네트워크·콜백 오류 안내.
- 최초 프로필: 기본 닉네임, 추천 닉네임, 계정 프로필 불러오기, 앨범 사진, 기본 이미지, 닉네임 2~20자 검증, 기본값으로 바로 시작.
- 로그아웃: 원격 세션 종료, 로컬 세션 삭제, 로그인 화면 복귀, 실패 안내.
- 회원탈퇴: 위험 안내 모달, 취소, 중복 실행 방지, Edge Function 호출, 로컬 세션 삭제, 실패 복구.

### 2. 홈

- 현재 동네 표시와 동네 패널 열기/닫기.
- 현재 위치 권한 요청, 행정동 탐지, 신규 동네 추가, 기존 동네 전환, 최대 5개 제한.
- 동네 제거 확인, 활성 동네 제거 후 다음 동네 선택, 기록 보존 안내.
- 냥냥단 사원증, 직급, 도감·배지 수치, 프로필 이미지 fallback.
- 오늘 본 고양이 기록 진입, 개인 도감 진입.
- 수집 진행률, 이번 주 기록, 다시 만남, 배지 통계.
- 대표 고양이 또는 최근 고양이 목록, 고양이 상세 진입, 빈 상태.
- 획득 배지 미리보기, 전체 배지북 진입, 배지 빈 상태.
- 알림함 진입과 새 알림 표시.

### 3. 내 도감

- 내가 수집한 고양이만 표시.
- 이름, 번호, 털색, 관계, 메모, 태그 검색.
- 전체·털색·희귀 필터와 검색 조합.
- 고양이 카드 상세 진입.
- 미수집 페이지 힌트와 제보 수 표시.
- 수집 없음, 검색 결과 없음 상태.

### 4. 고양이 상세와 수정

- 대표 사진 fallback, 도감 번호, 이름, 털색, 희귀도, 관계 단계, 만남 수.
- 희귀도 기준 모달, 관계 단계 기준 모달.
- 성격 태그, 메모, 만남 타임라인.
- 정보 수정 진입, 게시글 작성 진입, 정보 신고.
- 정보 수정: 이름 필수, 대표 사진 선택·교체·기본 이미지, 성격 태그, 메모.
- 털색·희귀도·만남 횟수 수정 제한 안내.
- 저장 중 중복 제출 방지, 권한·업로드·DB 실패 시 입력 유지와 재시도.

### 5. 촬영과 등록

- 동네 미설정 시 촬영 진입 차단.
- 카메라 권한 확인·요청·거부, 카메라 준비, 전면/후면 전환, 촬영, 재촬영.
- CatVision 탐지·누끼, 처리 진행 표시, 탐지 실패, 모듈 미지원, 처리 오류.
- 앨범 테스트 사진 선택, 사진 권한 거부, 선택 취소.
- 원본 사진 대체 흐름과 자동 분석 미사용 표시.
- 원본·누끼 Storage 업로드, 관찰 생성, 현재 동네 후보 RPC 생성.
- 후보 없음, 최대 5개 후보, 후보 사진·최근 기록 표시, 접근성 라벨.
- 동일 고양이 확인 모달, 취소, 기존 고양이 만남 기록 연결.
- 새 고양이 등록: 이름, 털색, 성격 태그, 현재 동네, 메모, 사진.
- 이웃 확인 요청: 이름 없이 털색·동네·메모·사진으로 미확인 제보 생성.
- 잘 모르겠어요: 관찰을 `uncertain`으로 해소하고 도감 복귀.
- 업로드·관찰·후보·등록·연결 단계별 실패 시 사용자 안내, 중복 제출 방지, 부분 업로드 정리.

### 6. 동네 도감

- 현재 동네에 연결된 고양이와 미확인 제보만 표시.
- 전체·내 도감·확인 필요 범위 전환.
- 이름, 털색, 특징, 구역 검색.
- 전체·털색·희귀 필터와 범위 조합.
- 고양이 상세 진입, 빈 상태, 촬영 진입.
- 동네 기여 랭킹 로딩, 상위 사용자, 내 순위, 빈 상태, 실패와 재시도.
- 게시판 최근 글 미리보기, 빈 상태, 실패 안내, 전체 게시판 진입.
- 동네 지도·게시판 빠른 전환.

### 7. 동네 지도

- 현재 동네 지역만 지도에 표시.
- Kakao Map 로딩, WebView 오류 시 지역 목록 fallback.
- 지역 선택, 선택 지역 고양이 수, 고양이 최대 3마리 미리보기.
- 하단 시트 접기/펼치기, 고양이 상세 진입, 지역 빈 상태.
- 동네 도감·게시판 전환과 촬영 진입.
- 지도 키 누락·허용 도메인 오류·HTTP 오류 시 앱 사용 가능 상태 유지.

### 8. 동네 게시판

- 현재 동네 원본명·정규화 지역명으로 공개 글 조회.
- 전체·목격·질문·도움·일상 필터.
- 로딩, 빈 상태, 불러오기 실패, 다시 시도.
- 게시글 카드: 주제, 제목, 본문, 작성자, 시간, 연결 고양이, 사진, 공감·댓글 수.
- 글쓰기 및 게시글 상세 진입, 도감·지도 전환.

### 9. 게시글 작성

- 주제 선택, 선택 제목, 2자 이상 본문 검증.
- 내 도감 고양이 연결·변경·해제, 연결할 고양이 없음 상태.
- 사진 다중 선택, 최대 5장, 권한 거부, 선택 취소, 개별 삭제.
- 게시 중 중복 제출 방지, 본문 유지, DB·Storage 실패 안내.
- 이미지 저장 실패 시 게시글과 이미지를 함께 롤백.
- 작성 취소 시 입력 손실 확인.

### 10. 게시글 상세와 관리

- 게시글 로딩, 찾을 수 없음, 실패, 재시도.
- 사진 목록, 연결 고양이 상세 진입.
- 공감 추가·취소와 실패 시 원상 복구.
- 댓글 작성, 빈 댓글 차단, 작성 실패 시 입력 유지.
- 본인 댓글 수정·저장·취소·삭제 확인과 실패 안내.
- 본인 게시글 수정: 주제, 제목, 본문, 사진 추가·삭제·교체.
- 본인 게시글 삭제 확인, 댓글·공감 행 cascade, Storage 사진 정리.
- 타 사용자에게 수정·삭제 액션 미노출 및 RLS 차단.

### 11. 내 게시글

- 내가 작성한 공개 활성 글만 조회.
- 주제 필터, 상세 진입, 바로 수정, 삭제 확인.
- 로딩, 빈 상태, 실패, 다시 시도, 새 글 작성.

### 12. MY와 프로필

- 사원증, 닉네임, 직급, 사번, 현재 동네, 수집·재발견·배지 통계.
- 직급 기준 모달과 다음 승급 진행률.
- 프로필 수정: 닉네임 2~20자, 추천 닉네임, 앨범 사진, 계정 사진, 기본 이미지.
- 획득 배지 미리보기와 전체 배지북.
- 우리 도감 주인공: 무료 1마리·냥꾸러미 3마리 선택, 해제, 순서 변경, 원자적 저장, 빈 상태.
- 근무 기록, 내 게시글, 알림 설정 진입.
- 로그아웃과 회원탈퇴.

### 13. 배지북과 근무 기록

- 전체·획득·미획득 필터, 배지 카테고리 필터.
- 최근 획득 배지, 다음 목표 배지, 획득 진행률.
- 배지 상세 모달, 획득 조건과 상태, 관련 화면 이동.
- 배지 데이터 없음 fallback 카탈로그.
- 근무 기록 목록, 고양이 상세 진입, 기록 없음 상태.

### 14. 알림함과 알림 설정

- 알림 이벤트 로딩, 당겨서 새로고침, 빈 상태, 실패 안내.
- 읽지 않음 수, 개별 읽음, 전체 읽음, 로컬 사용자별 읽음 상태.
- 고양이·촬영·도감·동네·MY 딥링크와 잘못된 대상 fallback.
- 알림 권한 확인·요청·거부, 기기 설정 이동.
- 탐험 리마인더 활성화와 18:00·20:00·21:30 선택.
- 동네 발견, 내 고양이, 배지와 직급, 기록 반응, 주간 리포트 토글.
- 로컬 예약과 원격 설정 동기화, 저장 실패 원상 복구.
- Expo push token 등록, 로컬 미리보기 알림, 실패 안내.

### 15. 공통 UI와 접근성

- 하단 탭 홈·내 도감·촬영·동네·MY 전환과 현재 탭 표시.
- 상세·작성·모달에서 하단 탭 노출 규칙.
- 작은 화면 스크롤, 키보드, 긴 닉네임·동네명·고양이명·게시글 텍스트 잘림.
- 버튼 최소 터치 크기, 아이콘 버튼 접근성 이름, 선택 상태 전달, 비활성 상태.
- 로딩 중 레이아웃 이동, 중복 탭 방지, 오류 후 재시도 가능 여부.
- 사용자 입력과 업로드 파일이 실패 시 유지되는지 확인.

| E-20 | 긴 텍스트와 작은 화면 | 겹침·잘림·버튼 밀림 없이 스크롤 가능 | 통과 |

`제한`은 처리 코드와 대체 UI까지 확인했지만 외부 OAuth 제공자, iOS 권한 영구 거부 상태, Kakao 장애를 이번 시뮬레이터 세션에서 실제로 만들지 못한 항목이다.

## 수정 및 재검증 결과

1. 앱 공통 데이터와 고양이 데이터에 로딩·오류·재시도 상태를 연결하고, 사용자 전환 중 오래된 요청 결과가 새 세션을 덮지 않도록 취소 가드를 추가했다.
2. 촬영 업로드, pending 관찰, 게시글 이미지 업로드에 보상 정리를 추가했다. 샘플 사진으로 후보 생성 후 취소했을 때 pending 관찰 `0`, 사용자 `cat-images` 객체 수가 기존 `1`로 복구됨을 확인했다.
3. 관찰 해소는 수정된 행을 반환받아 0행이면 실패 처리한다. 본인 pending 관찰 삭제 RLS도 추가하고 트랜잭션으로 검증했다.
4. 게시글은 숨김 상태 생성 후 이미지 연결 성공 시 활성화한다. 수정은 `update_community_post_with_images`, 이미지 교체는 `replace_community_post_images` RPC로 원자화했다.
5. 게시글·댓글 수정/삭제, 작성 중 이탈 확인, 사진 추가·삭제·교체, 내 게시글 바로 수정 경로를 구현하고 실제 댓글 생성·수정·삭제 후 잔여 데이터가 없음을 확인했다.
6. 대표 고양이는 `replace_featured_cats` RPC로 교체하고 entitlement를 DB에서 읽는다. 무료 2마리 저장은 DB에서 거부되고 1마리는 허용되는 것을 롤백 테스트했다.
7. 프로필 저장은 공개 컬럼 제한을 유지하면서 `upsert_own_profile` RPC로 누락 프로필까지 복구한다. 이미지 업로드 실패·DB 실패 시 새 파일과 Auth metadata를 원복한다.
8. 알림 설정과 읽음 상태는 실패 시 낙관 상태를 되돌린다. 영구 거부 시 기기 설정 이동을 제공하고 모든 스위치에 접근성 이름을 부여했다.
9. 상세·작성·설정 화면에서 하단 탭을 숨기고, 고양이 상세·배지북·알림·게시판의 진입 화면을 기억해 올바른 화면으로 돌아가도록 수정했다.
10. 배지 판정 마이그레이션을 원격 DB에 적용하고 기존 데이터를 보정했다. `단골 냥이`가 5/5 미획득으로 남던 상태가 획득으로 바뀌었으며 배지북은 15/26으로 다시 집계됐다.
11. 클라이언트 13종 털무늬와 DB 제약을 일치시켰다. 희귀도 함수의 지역 변수 충돌을 수정하고 `고등어냥` 생성 RPC를 트랜잭션에서 실행한 뒤 롤백했다.
12. 동네 리더보드는 인증 사용자 프로필의 공개 컬럼만 읽도록 제한하고 `security invoker`, 조회 기간·개수 상한을 적용했다.
